Cara Upload File

Berikut step by step lengkap untuk upload laman mu sendiri

# Ambil repo dari RWCLI
git clone https://github.com/rizkydaffy/rwcli.git
# Buka Repo RWCLI
cd rwcli
# Instalasi dependensi
pip install -r requirements.txt

# Penggunaan RWCLI
    Upload: python rwcli upload whenyh.html
    list: python rwcli list
    help: pythonn rwcli -h

# Perhatian Penggunaan
    perhatikan: file .html nya harus berada di folder yg sama dengan script nya
    perhatikan: Konten yang Anda unggah akan disimpan di server kami.
        Mengunggah file dengan nama yang sama akan menimpa versi sebelumnya

# Rules Penggunaan
    1. jangan naro web judi
    2. jangan naro shell
    3. jangan mainin burpsuite
    4. jangan naro file extensi lain, selain .html

# Kreativitas
    1. jikalau mau ngirim 2 file boleh, semisal rizky.html sama rizky.css (panggil di html)
    2. Boleh naroh css didalem html kalo males bikin .css
    3. boleh lah up file pura pura deface seterah (asal jan deface sistem nya ajg)
